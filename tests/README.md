# 回归测试说明

当前测试采用 `unittest`，目标是覆盖最基本的工程回归场景：

- 统一入口 `main.py --list` 可正常工作
- `plate_3d` 轻量配置可生成主结果文件
- `tip_plane` 轻量配置可生成轨迹与摘要
- `cross_field` 轻量配置可生成轨道与摘要
- `pinn` 轻量配置可完成训练并生成图表

运行方法：

```bash
python -m unittest discover -s tests -v
```

说明：
- 测试使用轻量缩小版参数，不追求物理精度，只验证代码主路径和关键输出是否正常。
- 若后续修改了模块入口、输出文件名或配置字段，应同步更新这些测试。
