# plate_3d Richardson / GCI 收敛报告（定稿）

## 1. 分析对象与方法

本次对 `plate_3d` 模块采用统一 refinement ratio 的三层网格组进行收敛分析。为避免粗搜索采样点直接污染 QoI，三层网格的 `J_lim` 均使用根附近再求解方式获得，即先用 continuation 给出种子，再在变号区间附近插值取根，并在该根点重新求解主场分布。

- 网格层级：[17, 25, 37]
- refinement ratio r21：1.500000
- refinement ratio r32：1.500000
- 是否统一 refinement ratio：是
- 安全系数 Fs：1.25

## 2. J_lim 的 Richardson / GCI 结果

- 观测收敛阶 p：3.679557
- Richardson 外推值：3.147112e+05 A/m²
- 细网格 GCI：0.835107 %
- 粗网格 GCI：3.629074 %
- 渐近区检查比：0.977498
- J_lim 随网格加密是否单调：是

从该结果看，三层网格下的 `J_lim` 已呈现正的观测收敛阶，并且渐近区检查比接近 1，说明当前三层网格组已经进入较合理的 Richardson / GCI 适用区间。细网格 GCI 可视为当前 `J_lim` 数值不确定度的一阶工程估计。

## 3. 阴极中心梯度的收敛结果

## 6. 结论

本轮收敛分析采用统一 refinement ratio 的三层高网格组，并通过根点再求解降低了粗 bracket 对 `J_lim` 的污染。结果表明：

1. `J_lim` 的三层结果随网格加密单调下降；
2. 观测收敛阶为正，且渐近区检查比接近 1；
3. 细网格 GCI 已可作为论文前期数值不确定度评估结果写入报告；
4. 目前外层自洽严格收敛标志仍未在三层根点算例上全部触发，因此若继续追求论文定稿级精度，仍建议进一步提高 fine-grid 外层迭代上限，并在最细层再追加一组更高网格进行复核。
