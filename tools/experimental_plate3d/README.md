# Experimental Plate3D Utilities

This directory archives one-off probe scripts used during the 37/55-layer warm-start debugging stage.
They are kept for traceability only and are **not** part of the active production workflow.

Do not call these scripts from `src/main.py`. The active entry point is `src/main.py` and the active workflow is `src/fdm_3d/workflow.py`.
